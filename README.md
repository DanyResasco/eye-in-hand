# eye-in-hand
Progetto che si occupa della visione di una camera monoculare montata su un braccio robotico. Questo braccio, montato su una carrozzina, viene utilizzato come aiuto per persone disabili per premere dei pulsanti. Il codice Ã¨ generale per qualsiasi pulsantiera. 
L'utente premendo sul tablet fornisce al codice il pulsante desiderato. Utilizzando degli algoritmi di riconoscimento delle forme e dei controlli i.e Kalman Filter, controlli in forza, riesce a portare il braccio nel punto desiderato.
